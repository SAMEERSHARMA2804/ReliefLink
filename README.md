ReliefLink is a platform designed to connect individuals with essential emergency resources. The system enables users to add, view, update, and manage resources such as food, shelter, and medical assistance through a structured and user-friendly interface.

🔧 Key highlights:

Backend developed using Django (MVT architecture)
Implementation of complete CRUD operations
User authentication (login/signup functionality)
Seamless integration between frontend and backend
Focus on usability and responsive design
